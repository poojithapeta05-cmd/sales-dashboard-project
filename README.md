# Sales Data Analysis Project

## 📌 Project Overview
This project analyzes sales data using Python. It focuses on understanding sales performance, identifying trends, and generating insights using data visualization.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib

 📊 Features
- Data cleaning and preprocessing
- Sales analysis by region
- Product-wise performance analysis
- Data visualization using bar chart, pie chart, and line chart

📈 Output
- Bar Chart: Sales by Region
- Pie Chart: Sales by Product
- Line Chart: Daily Sales Trend

📁 Files Included
- main.py
- sales_data.csv
- cleaned_sales_data.csv
- Chart images (PNG files)

🚀 How to Run
1. Install required libraries:
   pip install pandas matplotlib
2. Run the script:
   python main.py
